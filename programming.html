В IT-академии проводятся занятия по Scratch-программированию, Java для начинающих, web-программированию и компьютерной графике

Курсы по секции:

  • Английский для IT (5-7 классы)

  • Алгоритмизация и программирование (8-11 кл., для начин.)

  • Виртуальная и дополненная реальность (7-8 классы)

  • Компьютерная графика и веб-дизайн (6-7 классы)

  • 3D-моделирование и 3D-печать (8-10 классы)

  • Scratch-программирование (3-4 классы, для начинающих)

  • Scratch-программирование (3-4 кл., для продолжающих)

  • Scratch-программирование (5-6 классы, продвинутый уровень)

  • Web-программирование (7-9 классы, для начинающих)

  • Web-программирование (8-11кл., для продолжающих)

  • Java для начинающих (8-11 классы)

  • Java для продолжающих (9-11 классы)

  • Python для начинающих (6-7 классы)

  • Python для продолжающих (разработка игр, 7-9 классы)

Информатика и программирование

Направления занятий в рамках секции информатики и программирования включают:

  1. Scratch-программирование
    Этот курс предназначен для самых молодых учеников TI-академии и нацелен на изучение основ алгоритмизации и программирования с использованием языка Scratch.
  2. Java для начинающих
    Язык программирования Java является одним из наиболее популярных и распространенных в мире на сегодняшний день. В рамках этого курса ученики ознакомятся с синтаксисом Java и изучат способы его применения для решения задач.
  3. Web-программирование
    Данный курс посвящен изучению таких технологий, как HTML, CSS и JavaScript, которые используются при разработке веб-сайтов и клиентской части интернет-приложений.
  4. Компьютерная графика и web-дизайн
    Этот курс будет полезен всем, кто интересуется художественным творчеством и хочет использовать компьютер как инструмент для воплощения своих художественных идей.

Кроме того, учащиеся IT-академии имеют возможность проявить себя, принимая участие в различных мероприятиях и соревнованиях по информатике и программированию как республиканского так и международного уровня.
